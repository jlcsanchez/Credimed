# Credimed — Archived Service Agreement Versions

This directory preserves prior versions of the Credimed Service Agreement
for record-keeping. Per Section 1 of the current canonical Agreement
(`/legal/AGREEMENT_v1.9.html`):

> "This version supersedes prior versions for all claims initiated on or
> after the Effective Date. Claims initiated under prior versions of this
> Agreement continue to be governed by the version in effect at the time
> of acceptance."

Archived versions:

- `AGREEMENT_v1.0.md` — original draft
- `AGREEMENT_v1.1.md` — first revision
- `AGREEMENT_v1.2.md` — second revision (last `.md` version that was
  previously linked from the live site)

Versions v1.3 through v1.8 were intermediate iterations during the
April 2026 counsel-prep review and were never published live; they are
not preserved in this archive.

The current canonical Agreement is at `/legal/AGREEMENT_v1.9.html`.

**Do not edit these files** — they are immutable historical records of
what Members signed under prior agreements. Any updates go in the
canonical file.
