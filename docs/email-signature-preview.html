Install in Gmail (3 minutes):
1. In this preview, click and drag from "Juan Luis Sanchez" through the HIPAA confidentiality line at the bottom to select the entire signature block (incluyendo el footer chiquito).
2. Copy (Cmd+C on Mac, Ctrl+C on Windows).
3. Gmail → gear icon (top right) → See all settings → scroll to Signature → Create new → name it "Credimed".
4. Paste (Cmd+V / Ctrl+V) → set as default for "New emails" and "Reply/forward" → tick "Insert signature before quoted text in replies" → Save changes at the bottom.
5. Compose a new email to confirm it renders correctly.
